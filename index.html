Введение

Чтобы начать понимать Snake Script, много ума не надо. Для этого достаточно выучить функции этого ЯП.

Синтаксис

Синтаксис в языке достаточно простой, пишешь название функции, и в конце: ";"

Переменные

В нашем языке есть такие переменные как: path, int, string. path - Тот же string, но который используется для обозначение путей, и умеет обрабатывать: "\", int - Переменная которая хранит в себе целые числа, например: int a = 1;, string - Переменная которая хранит в себе текст.

Класс System

Имеет в себе такие функции как:

System.print("текст"); - Функция которая выводит текст, принимает аргументы: string, path.

System.BSOD(); - Вызывает BSOD, (Blue Screen Of Death).

System.wait(число); - Ожидает перед выполнением следующей строки, (В миллисекундах)

System.setLocale = Locale.RU; - Устанавливает русский язык для вывода в консоль

System.execute("комманда"); - Выполняет указанную комманду в консоли, например tasklist

Класс Script

Имеет в себе такие функции как:

Script.end(); - Обрывает скрипт, и его выполнение заканчивается

Класс Network

Имеет в себе такие функции как:

Network.downloadFile("источник", "путь в который будет установлен файл"); - Скачивает файл из ссылки, до тех пор пока не скачает, после скачивания продолжит скрипт.

Network.downloadFileAsync("источник", "путь в который будет установлен файл"); - Скачивает файл из ссылки, и в это же время продолжает выполнять скрипт.

Класс Render

Render.msgBox("текст", "название окна"); - Выводит на экран MessageBox с указанным текстом, и названием окна

Примеры

Вызов BSOD:

System.BSOD();

Вывод текста:

System.print("тут текст");

Вывод текста, и установка файла:

System.print("Устанавливаем файл...");

Network.downloadFile("www.example.com/file.exe", "C:\Users\zmeyskiy\Downloads\file.exe");

System.print("Файл установлен!");
